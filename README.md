<h1 align="center">NASDAQ 보물주 발굴기</h1>
<p align="center">나스닥·NYSE 전종목 6,000+개를 매일 4회 자동 스캔 — <strong>"아직 아무도 모르는 종목"</strong>을 기술 지표로 먼저 찾습니다</p>

---

## 왜 만들었나?

개인 투자자는 하루에 수천 개 종목을 직접 볼 수 없습니다.
나스닥 3,500개 + NYSE 2,500개 = 6,000개 이상. 이 중에는 이미 차트에 신호가 쌓이기 시작했지만 아직 주목받지 못한 종목이 분명히 존재합니다.

이 프로젝트는 그 탐색 과정을 자동화합니다.

```
"골든크로스가 뜬 종목"은 모두가 안다.
우리는 골든크로스가 뜨기 2~4주 전을 본다.
```

## 스윙 매매 정보 (ATR 기반 자동 산출)

모든 목표가·손절가는 종목별 **실제 변동성(ATR)** 에 맞게 자동 계산됩니다.

```
진입가    = 현재가 ± 0.5%
손절가    = 현재가 − 2 × ATR          (리스크 제한)
1차 목표  = 현재가 + 3.5 �� ATR        (3−4주 목표)
2차 목표  = 현재가 + 6 × ATR          (6−8주 목표)
손익비    = 목표 수익% ÷ 손실%   →  2:1 이상 권장
```

> $10짜리 주식과 $500짜리 주식에 같은 % 기준을 쓰는 것보다
> 종목별 변동성에 맞춘 ATR 기준이 훨씬 현실적입니다.
